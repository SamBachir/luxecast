# Workflows

## Talent Workflow
Register -> Verify -> Build Profile -> Subscribe -> Admin Approval -> Go Live
-> Receive Booking/Casting Call -> Accept -> Complete Job -> Payment Captured
-> Commission Deducted -> Payout Released -> Review Updates Ranking

## Client Workflow
Register -> Add Payment Method -> Search/Post Casting Call -> Select Talent
-> Confirm Booking -> Payment Secured -> Job Completed -> Review -> Rebook

## Admin Workflow
Approve Talent -> Monitor Marketplace -> Control Payments -> Resolve Disputes
-> Enforce Policies -> Track Revenue and Platform Health
